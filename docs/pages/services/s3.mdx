# Amazon S3

## Installation

Install package

```sh
yarn add @uplo/service-s3
```

Add service to Uplo

```ts
import S3Service from '@uplo/service-s3'

const uplo = Uplo({
  services: {
    s3: new S3Service({
      isPublic: false,
      region: process.env.AWS_REGION,
      bucket: process.env.AWS_BUCKET,
      accessKeyId: '*****',
      secretAccessKey: '*****',
    }),
  },
})
```
